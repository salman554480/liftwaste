NOTIFICATION SOUND SETUP

1. Download a notification sound MP3 file (free options available at):
   - https://notificationsounds.com/
   - https://www.zapsplat.com/
   - https://freesound.org/

2. Rename it to "notification.mp3" and place it in this assets folder

3. The notification system will automatically use this file to play sounds when new pending emails arrive
